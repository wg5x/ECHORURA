# ECHORURA 项目文档

本目录用于整理 ECHORURA 当前网站、BP、垂直化产品方向和市场调研材料。

## 文件说明

- `极声音乐BP2.0版.pptx`：原始 BP 文件。
- `docs/网站与BP审阅总结.md`：对现有网站和 BP 的审阅总结，包含问题、对齐建议、商业模式分析和分工建议。
- `docs/垂直化打法报告.md`：围绕“视觉障碍与低操作门槛人群”的垂直化战略打法，包含产品定位、Web3 角色、MVP 和路线图。
- `docs/市场调研报告.md`：面向该垂直方向的市场调研，包含目标用户、竞品地图、市场空白、进入策略和验证指标。

## 当前核心方向

建议将 ECHORURA 从“大而全的 Web3 音乐经济系统”收敛为：

> 面向视觉障碍与低操作门槛人群的语音 AI 音乐创作工具，用 Web3 记录模板来源、二创链路和分成关系。

短期重点不是完整音乐平台，而是验证：

- 用户是否能通过语音完成 30-60 秒音乐创作。
- 视觉障碍用户是否能在低屏幕依赖下完成创作、修改和发布。
- 模板复用和二创链路是否能自然发生。
- 用户或机构是否愿意为生成、模板、无障碍创作工具或试点服务付费。
