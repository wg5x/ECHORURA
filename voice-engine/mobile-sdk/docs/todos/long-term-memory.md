# TODO: 用户长期记忆

## 目标

把通话报告中的稳定事实、偏好、互动方式和待承接话题压缩成可控记忆卡，下次通话按开关注入。

## 第一版范围

- 复用现有 `Memory Card`。
- 新增 `facts` 字段，承载比偏好更稳定的长期事实。
- 继续过滤敏感身份、住址、健康、财务、未成年人等高风险内容。

## 验收

- “我主要做产品设计”可进入 `facts`。
- “我喜欢短一点”可进入 `preferences`。
- 敏感内容不会进入长期记忆。
- 旧记忆卡缺少 `facts` 时可正常兼容。

## 后续

- 用户可逐条删除和确认记忆。
- 增加来源会话和置信度。
- 支持跨场景合并和按场景隔离。
