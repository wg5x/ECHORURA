# TODO: 对话意图识别

## 目标

在每段用户文本进入业务编排前，先给出一个可解释的本地意图分类结果，用于退出、切换场景、记忆写入、播客生成和风险拦截等后续流程。

## 第一版范围

- 使用确定性规则识别少量高价值意图。
- 识别广告机会意图，但只输出分类结果，不直接触发广告投放。
- 返回 `intent`、`confidence`、`reasons`、`actions`。
- 不直接调用大模型，不替代上游安全审核。

## 验收

- 空文本返回 `unknown`。
- “先这样吧，结束通话”识别为 `exit_session`。
- “把这份文档生成播客”识别为 `podcast_request`。
- “记住我喜欢短一点的回答”识别为 `memory_update`。
- “附近有没有适合亲子游的酒店套餐”识别为 `ad_opportunity`。
- 有单元测试覆盖主要规则。

## 后续

- 接入实时 ASR 后的事件流。
- 增加场景相关意图，例如设备控制、课程练习。
- 允许管理员查看意图命中统计。
